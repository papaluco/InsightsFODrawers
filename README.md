# District ENP Dashboard

A specialized performance tracking dashboard designed to monitor and analyze **Eligibility Not Provided (ENP)** metrics across the school district.

## 📊 Project Overview
This dashboard provides a high-level summary of ENP data, allowing administrators to compare actual performance against district benchmarks. It features:
* **ENP Overview**: A quick-glance summary of total enrollment, non-program meal counts, and benchmark variances.
* **Eligibility by Meal Type**: A detailed breakdown of meal service categories (Breakfast, Lunch, Snack, etc.) and their respective eligibility statuses.
* **Details by School**: A searchable, sortable grid providing granular data for every school in the district.

## 🚀 Technical Stack
* **Framework**: React with TypeScript
* **Styling**: Tailwind CSS
* **Icons**: Lucide React
* **Components**: Custom-built modular UI components
